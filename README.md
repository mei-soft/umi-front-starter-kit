
<h1 align="center">Starter Frontend</h1>

## Usage

### Use bash

```bash
$ git clone https://github.com/mei-soft/umi-front-starter-kit
$ cd umi-front-starter-kit
$ npm install
$ npm start         # visit http://localhost:3000
```

### Use by docker

```bash
# preview
$ docker-compose up
# open http://localhost:8000
```
